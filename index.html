Google Channel Manager 查看源码

功能特点

  • 批量添加 Google 渠道
  • 支持手动输入渠道名称列表或自动生成渠道名
  • 自动选择产品类型
  • 实时进度显示
  • 失败重试机制（最多3次重试）
  • 详细的成功/失败状态报告

使用方法

1. 在输入框中粘贴渠道名称列表（每行一个）
   或设置需要生成的渠道数量

2. 选择产品类型

3. 点击"获取数据"按钮开始处理

// 处理过程会显示：
- 实时进度
- 成功添加数量
- 失败的渠道列表（如果有）

Google Tags Checker 查看源码

功能特点

  • 检测 GA4（Google Analytics 4）的安装状态
  • 显示 GA4 的 Measurement ID
  • 检测 Google Publisher Tag (GPT) 广告位
  • 检测 Google AdSense 代码
  • 显示广告位容器和相关代码
  • 支持一键复制检测到的代码

使用方法

1. 在需要检查的网页上点击插件图标
2. 插件会自动检测并显示：
   - GA4 安装状态和 ID
   - 页面上的广告位数量
   - 每个广告位的详细信息
   
3. 可以通过"复制代码"按钮复制任意广告代码

Yandex Game Details 查看源码

功能特点

  • 自动获取 Yandex 游戏信息
  • 提取游戏图标（多种尺寸）
  • 获取游戏 Banner 图
  • 获取游戏基本信息：
    • 游戏标题
    • 游戏类型
    • 游戏说明
    • 游戏源地址
  • 自动处理游戏菜单交互
  • 支持图片预览功能

使用方法

1. 在 Yandex 游戏页面上点击插件图标
2. 点击"获取数据"按钮
3. 插件会自动：
   - 打开游戏菜单
   - 收集游戏信息
   - 显示完整的游戏详情，包括：
     * 游戏图标和预览图
     * 游戏基本信息
     * 详细说明文本

注意：仅支持 Yandex 游戏网站使用
